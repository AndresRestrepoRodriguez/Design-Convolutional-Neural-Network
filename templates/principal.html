Bienvenido a DeCNN


Esta aplicación proporciona un entorno Web intuitivo para el diseño, entrenamiento, evaluación y descarga de modelos de Redes Neuronales por Convolución (CNN) haciendo uso de Keras, una biblioteca de red neuronal de código abierto escrita en Python



Dentro de la aplicación, se encontrará una zona de diseño de la CNN, una zona de visualización del Modelo, una zona de evaluación mediante las Curvas de ROC, una zona de Predicción para probar el modelo y una zona de descarga para descargar el modelo y un compendio de datos

Curvas de ROC

Una curva ROC (curva de característica operativa del recepto) es un gráfico que muestra el rendimiento de un modelo de clasificación en todos los umbrales de clasificación. Esta curva representa dos parámetros: Tasa de verdaderos positivos y Tasa de falsos positivos
